Visualizaciones de kernels

Este proyecto nace por la necesidad de encontrar una interpretación visual de los kernels más comunes en la actualidad, los kernels son herramientas que cambian la dimensionalidad de los datos para realizar mejores clasificaciones, regresiones o clasificaciones.

Las Tecnologías utilizadas son Plotly Dash y heroku para mostrar los gráficos en la web, los datos son generados por grillas y son procesados en python con módulos como Numpy y Pandas.

link proyecto: kernels-visualizations

link Repo: GitHub


Ke horario

Que horario, es mi proyecto más largo ha cambiado varias veces de rumbo y su nombre no significa nada realmente, es mi startup, mi proposito es solucionar algun problema con las tecnologías que conozco.

Las tecnologías utilizadas son Django y Heroku para tener una interfaz web, Pandas, Numpy y Postgres SQL para el manejo de datos.

En un futuro planeo integrar algoritmos de ML supervisado para recomendaciones personalizadas según los gustos de los usuarios y algoritmos de ML no supervisado para determinar los distintos tipos de segmentos que tenga en mi aplicación.
